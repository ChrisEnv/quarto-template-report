5  Interactive Content with Widgets

We use R packages like plotly and leaflet to create interactive content.

library(palmerpenguins)
data(penguins, package = "palmerpenguins")

plot_ly(data = penguins, x = ~bill_length_mm, y = ~bill_depth_mm, type = 'scatter', mode = 'markers')
Warning: Ignoring 2 observations

Figure 5.1 shows a scatter plot of bill length vs. bill depth in penguins. You can hover over the points to see the data values. It is created using the plotly package.

5.2 Leaflet Map

library(leaflet)

# Placeholder data; replace with actual geographic data if available
leaflet() %>% 
  addTiles() %>% 
  addMarkers(lng = 174.768, lat = -36.852, popup = "Auckland")

Figure 5.2 shows a sample map with a marker at Auckland, New Zealand. It is created using the leaflet package. Note that interactive features do not display in PDF.
